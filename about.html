website mẫu

nhận định về ưu điểm

  1. có thể dùng bootstrap chia được layout
  2. có thể làm các style đơn giản
  3. đã biết cách deploy web tĩnh lên github
  4. làm được một số hiệu ứng đơn giản như chuyển màu, hover đổi màu,....

nhận định về nhược điểm

  1. chưa rõ ràng trong cách bố trí layout
  2. chưa làm được các style nâng cao cũng như các hiệu ứng ở mức trung bình
  3. response chưa tuân thủ hoàn toàn thiết kế
  4. chưa sử dụng nhuần nhuyễn scss (mixin, extend,...)
  5. cách bố trí file css chưa tốt
  6. tỉ lệ hoàn thiện website kém khoảng 30 - 40%
  7. chưa vận dụng được js vào trong web
